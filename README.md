## Waltz 🐶
My first project with four team members.

#### 개발목표
요구사항 분석과 계획 등 프로젝트 전반의 개발주기 이해 및 Spring framework 기초 이해도 향상, 팀원들과의 협업을 통한 웹페이지 구축 

#### 사용기술
* Java
* Spring framework
* MyBatis
* HTML
* CSS
* JavaScript
* Oracle

#### 주요기술
* 회원가입, 회원정보변경, 회원탈퇴
* 게시판 글 작성, 수정, 삭제
* 다중 파일 업로드
* 인터셉터를 통한 페이지 이동 및 권한부여
* 계산툴을 이용한 비만도, 나이, 칼로리 계산

#### 개선사항
* 이미지 로딩 속도 해결 필요
* 모니터 화면비율에 따른 여백 조정
